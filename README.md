## ⚙️ Troubleshooting Common Issues 

If you have trouble with the game, check these frequent solutions. 

### The game does not launch
Verify that your antivirus software did not block the file. Sometimes security tools prevent new games from starting. Check your antivirus history to see if the game executable is on a block list. Add the game folder to your exclusion list to fix this problem.

### Audio sounds distorted
Check your audio output device in the Windows Sound Settings. Ensure your device is set to stereo or the correct configuration for your speakers. Update your audio drivers if the problem persists.

### The game runs slowly
Reduce the visual settings inside the game menu. Lowering the shadows or turning off anti-aliasing provides a smoother frame rate on older hardware. Close other open applications before you start the game to free up your computer memory. 

### Screen looks blurry
Ensure the game resolution matches your monitor's native resolution. Open the settings menu, go to the Video tab, and select the resolution that matches your screen display.
